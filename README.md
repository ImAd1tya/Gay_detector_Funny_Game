#  Gay Detector (Java Swing Fun App)

A funny interactive desktop app built using **Java Swing**.
This project demonstrates GUI design, event handling, animation (moving button), custom fonts, gradient background, and sound effects.

⚠️ This is created for fun and learning purposes only.

---

## 🚀 Features

* 🖥 Custom GUI using Java Swing
* 🎨 Gradient romantic background (LightPink → Plum)
* 🔤 Custom fonts (Comic Sans MS, Papyrus, Arial)
* 🎭 Moving "NO" button effect
* 🔊 Sound effect when hovering on "NO"
* 💬 Popup result message

---

## 🛠 Technologies Used

* Java
* Swing (javax.swing)
* AWT (java.awt)
* Event Listeners
* Java Sound API

---

## 📂 Project Structure

```
Detector.java
no_sound.wav
README.md
```

Make sure `no_sound.wav` is inside the project root folder.


---

## 🎯 Learning Concepts Used

* OOP structure
* JFrame & JPanel
* Custom painting using Graphics2D
* MouseListener & ActionListener
* Random positioning logic
* Audio playback with Clip
* GUI event-driven programming
